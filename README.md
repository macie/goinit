# goinit

**goinit** initialize Go project with Git repository.

## Usage

```sh
$ ./goinit
```

## Install

Using `curl`:

```sh
curl -fLO https://raw.githubusercontent.com/macie/goinit/dev/goinit
chmod +x goinit
```

or with `wget`:

```sh
wget https://raw.githubusercontent.com/macie/goinit/dev/goinit
chmod +x goinit
```
